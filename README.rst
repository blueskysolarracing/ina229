======
INA229
======

INA229 is a Python driver for Texas Instruments INA229 85-V, 20-Bit,
Ultra-Precise Power/Energy/Charge Monitor With SPI Interface.

Features
--------

- High-level hardware usage.
- Low-level hardware usage.
